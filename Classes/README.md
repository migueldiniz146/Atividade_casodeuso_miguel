# Acrônimos usados neste diagrama de classes

## Classes
- USR → Usuário
- ALN → Aluno
- ADM → Administrador
- MAT → Matéria
- QZ → Quiz
- QUE → Questão
- ALT → Alternativa
- RES → Resultado

## Atributos
- IDU → id_usuario
- IDM → id_materia
- IDQ → id_quiz
- IDQUE → id_questao
- IDALT → id_alternativa

## Métodos
- LOGIN() → fazerLogin()
- REG() → criarConta()
- VMAT() → visualizarMaterias()
- RQUIZ() → responderQuiz()
- VRES() → visualizarResultado()
- CMAT() → cadastrarMateria()
- CQUE() → cadastrarQuestao()
- EDIT() → editarConteudo()
- LQUE() → listarQuestoes()
- IQZ() → iniciarQuiz()
- FQZ() → finalizarQuiz()
- MQUE() → mostrarQuestao()
- ERES() → exibirResultado()
