# Atividade - Casos de Uso (Miguel)

Este repositório contém a entrega individual da atividade **GitHub Individual (Repositório)**.

## Estrutura
- **CasosDeUso/** → Documentação dos casos de uso atribuídos.
- **Classes/** → Diagramas de classes (Interface, Negócio, Controle).
- **Sequencia/** → Diagramas de sequência por caso de uso.
- **Acronimos.md** → Lista de símbolos e acrônimos utilizados.

## Objetivo
Organizar e apresentar de forma clara os artefatos do projeto, seguindo as instruções da atividade:
1. Casos de Uso  
2. Classes por Caso de Uso  
3. Sequência por Caso de Uso  
4. Acrônimos (símbolos)
